EJECUCIONES

- VERSION PHP
  1. EJECUTAR EL ARCHIVO /server/crearBD.php EN EL NAVEGADOR EL ARCHIVO PARA LA CREACION DE LA BASE DE DATOS
  2. EJECUTAR EL ARCHIVO /server/create_user.php EN EL NAVEGADOR EL ARCHIVO PARA LA INSERCION DE USUARIOS DE PRUEBA
  3. PROBAR EL LOGIN CON UNO DE LOS USUARIOS INGRESADOS EN LA BD

- VERSION NODEJS
  1. EJECUTAR mongod DENTRO DE LA CONSOLA
  2. DENTRO DE LA CONSOLA, UBICARSE EN EL DIRECTORIO DEL PROYECTO
  2. EJECUTAR /server/index.js PARA EL CAMBIO DEL PUERTO DE ESCUCHA (3000), CREACION DE LA BASE DE DATOS Y USUARIO DE PRUEBA
  3. PROBAR EL LOGIN CON EL USUARIO DE PRUEBA CREADO: username: user1,password:12345
